# SmartTour - Quick Start Guide

## 🚀 How to Use SmartTour

### Starting the Application

1. **Open Terminal** and navigate to the project:
```bash
cd c:\Users\HP\Desktop\smartTour\smarttour-test
```

2. **Start the development server**:
```bash
npm run dev
```

3. **Open your browser**:
```
http://localhost:3000
```

You should see the SmartTour homepage with 8 navigation tabs.

---

## 📊 Dashboard - Your Starting Point

When you first load SmartTour, you're on the **Dashboard** tab.

### What You Can Do Here:
✨ See descriptions of all 6 AI agents
🚀 Click **"Plan Your Journey Now"** button to generate a complete trip itinerary
📅 View your generated 3-day travel plan
👨‍🏫 See recommended guides for your trip

**Try It Now**: Click the big blue button to plan your first journey!

---

## 🗺️ Map Explorer - Discover Places

Navigate to the **Map Explorer** tab to find places near you.

### Features:
- 🏨 **Hotels** - Find accommodations
- 🍽️ **Restaurants** - Discover local food
- 📍 **Places** - Explore attractions
- 🌤️ **Weather** - Check conditions

### How to Use:
1. Click any category button (Hotels, Restaurants, etc.)
2. Grant location permission when prompted
3. See nearby places displayed in results
4. Click on a result to see more details

**Example**: Click **Map Explorer** → Click 🏨 Hotels → Allow location → See hotels near you!

---

## 📅 Itinerary - Your Travel Plan

The **Itinerary** tab shows your personalized daily schedule.

### What You'll See:
- Daily activities planned
- Breakfast, lunch, dinner suggestions
- Estimated distances and times
- Activity recommendations

### How to Generate:
1. Go to **Dashboard**
2. Click **"Plan Your Journey Now"**
3. Return to **Itinerary** tab
4. See your customized 3-day plan

## 👨‍🏫 Guides - Meet Your Local Experts

The **Guides** tab shows recommended local guides.

### Guide Information:
- 👤 Guide name and experience

- 🌐 Languages spoken
- 🎯 Specialties and interests
- 💰 Daily rate
- ✅ Compatibility score with your preferences

### How to Book:
1. Go to **Dashboard** and plan journey
2. Visit **Guides** tab
3. See top 6 recommended guides
4. Review their profiles

## 🌐 Translate - Overcome Language Barriers

The **Translate** tab helps you communicate in different languages.

### Features:
- 🔄 Real-time text translation
- 🌍 Multiple language support
- 🗣️ Common phrases library
- 🎯 Language detection
- 📱 Offline phrases available

### Languages Supported:
- English (Default)
- Hindi (हिंदी)
- Malayalam (മലയാളം)
- Spanish, French, and more

### How to Translate:
1. Click **Translate** tab
2. Click **"🌐 Translate Sample"** button
3. See English → Hindi translation
4. View common phrases grid
5. Use phrases while traveling

### Common Phrases Included:
- "Hello" - नमस्ते / നമസ്കാരം
- "Thank You" - धन्यवाद / ധന്യവാദം
- "Help Me" - मेरी मदद करें / എനിക്ക് സഹായിക്കുക
- "Where is...?" - कहाँ है...? / എവിടെയാണ്...?
- And many more!

## 🧭 Directions - Get Navigation

The **Directions** tab provides turn-by-turn navigation.

### Features:
- 🛣️ Multiple route options
- 📏 Distance calculation
- ⏱️ Travel time estimation
- 🚗🏍️🚶 Multiple travel modes
- 📍 Turn-by-turn instructions

### Travel Modes:
- 🚗 Car
- 🏍️ Bike/Motorcycle
- 🚶 Walking

### How to Get Directions:
1. Click **Directions** tab
2. Enter start location (or use default: Kochi)
3. Enter end location (or use default: Bangalore)
4. Select travel mode (Car/Bike/Walking)
5. Click **"Get Directions"**
6. View route summary and instructions

### What You'll See:
- Total distance
- Estimated travel time
- Turn-by-turn instructions
- Alternative routes (if available)

## 💡 Tips & Tricks

### Maximize Your Experience

1. **Start with Dashboard**
   - Always plan your journey first
   - This generates personalized recommendations

2. **Use Map Regularly**
   - Discover new places while exploring
   - Save your favorite locations

3. **Check Weather Frequently**
   - Plan activities around weather
   - Pack appropriate clothing

4. **Learn Key Phrases**
   - Write down emergency phrases
   - Practice pronunciation
   - Show translator to locals if needed

5. **Save Emergency Numbers**
   - Screenshot safety info page
   - Add contacts to phone
   - Share with travel group

6. **Book Guides in Advance**
   - Popular guides book up quickly
   - Book as soon as you see a good match
   - Confirm a day before

7. **Use Different Travel Modes**
   - Try walking to explore
   - Use bike for moderate distances
   - Use car for long trips

## 🆘 Troubleshooting

### "Map not showing properly"
- Allow location permissions
- Try refreshing the page
- Check internet connection

### "No restaurants/hotels found"
- Enable location services
- Try a different category
- Make sure you're zoomed in on map

### "Translation not working"
- Check internet connection
- Try shorter text
- Verify language selection

### "Can't book a guide"
- Ensure journey is planned first
- Check guide availability
- Try another guide

### "API endpoints not responding"
- Restart the development server
- Clear browser cache
- Check internet connection
